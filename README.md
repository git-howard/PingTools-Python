# 网络Ping扫描工具

一个基于Python和Tkinter的图形化网络Ping扫描工具，可以快速扫描指定IP范围内的主机连通性。

## 功能特点

- 支持多种IP输入格式：
  - CIDR网段格式（如：192.168.1.0/24）
  - IP范围格式（如：192.168.1.1-192.168.1.100）
  - 单个IP地址（如：192.168.1.1）
- 可调节并发扫描数量（1-200）
- 图形化显示扫描结果，16×N的格子布局
- 不同颜色表示不同状态：
  - 灰色：未开始扫描
  - 绿色：Ping通
  - 红色：Ping不通
- 实时显示扫描进度和结果统计
- 自动保存上次扫描的网段和设置到INI文件
- 鼠标悬停显示完整IP地址

## 使用方法

1. 运行程序：
   ```
   python ping_tool.py
   ```

2. 在"IP范围/网段"输入框中输入要扫描的IP范围：
   - 网段格式：`192.168.1.0/24`
   - 范围格式：`192.168.1.1-192.168.1.100`
   - 单个IP：`192.168.1.1`

3. 设置并发量（默认为50）：
   - 较小的并发量：扫描较慢但占用资源少
   - 较大的并发量：扫描较快但占用资源多

4. 点击"开始扫描"按钮开始扫描

5. 扫描过程中可以点击"停止扫描"按钮中断扫描

6. 扫描结果会以格子形式显示，每个格子代表一个IP地址

7. 鼠标悬停在格子上可以查看完整的IP地址

## 配置文件

程序会自动创建`ping_tool_config.ini`配置文件，保存以下设置：
- 上次输入的IP范围
- 上次设置的并发量

## 系统要求

- Python 3.6+
- Tkinter（通常随Python一起安装）
- Windows操作系统（ping命令基于Windows格式）

## 注意事项

- 扫描大量IP地址时，请适当调整并发量以平衡速度和系统资源占用
- 某些网络环境可能限制ICMP包，导致无法准确判断主机状态
- 程序需要管理员权限才能在某些网络环境中正常工作

## 示例

扫描局域网中的所有设备：
1. 输入IP范围：`192.168.1.0/24`
2. 设置并发量：`100`
3. 点击"开始扫描"
4. 查看扫描结果，绿色格子表示在线设备，红色格子表示离线设备
